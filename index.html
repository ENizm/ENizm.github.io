Темы курсовых работ

Курсовая работа №1. Аппроксимация заданного набора данных функцией вида Y=ax^b

Целью курсовой работы является разработка программы для аппроксимирующей функции в виде степенной функции Y = ax^b для набора не равноотстоящих точек.

Курсовая работа №2. VR-тренажер для нефтегазовой отрасли в режиме мультиплеер

Для отработки действий бригады в условиях ГНВП разработан виртуальный тренажер в режиме мультиплеера. Все члены бригады имеют возможность совместно отработать правильную последовательность действий при ликвидации ГНВП в условиях максимально приближенных к реальным, но без человеческих жертв и потерь ценного сырья.

Курсовая работа №3. Проектирование и создание автоматизированной базы данных «Государственная инспекция безопасности дорожного движения»

Целью курсового проекта является создание и заполнение данными базу данных в Microsoft SQL Server Management Studio и разработка консольного программного приложения в Microsoft Visual Studio.

Задачи

Аппроксимация

  • Построить график этой функции
  • Рассчитать коэффициент детерминации
  • Рассчитать среднюю ошибку аппроксимации

VR-тренажер

  • Создание 3D-модели превентора
  • Скелетная и инверсная анимация персонажей и техники
  • Покадровая анимация

База данных

  • проектирование ER-диаграммы
  • создание базы данных в программной среде Microsoft SQL Server Management Studio
  • разработка программного приложения к базе данных в Microsoft Visual Studio
